# Ayvy TCC - Layout

Um projeto de site estático simples para um conceito de restaurante/cafeteria chamado Ayvy. Este repositório contém layout HTML e CSS básico com páginas de navegação (Home, Sobre, Menu, Avaliações, Contato).

## 🔧 Estrutura do projeto

- `index.html`: Página principal com menu de navegação e banner.
- `style.css`: Estilos globais e variáveis CSS.
- `img/`: Imagens usadas no layout (por exemplo `img-curta-ayvy.jpeg`).
- `.gitignore` não presente (podem adicionar arquivos de build ou sistema local se necessário).

## ▶️ Executar localmente

1. Clone o repositório:
   `git clone <URL-do-repositório>`
2. Abra `index.html` no navegador (duplo clique ou via Live Server do VS Code).
3. Navegue pelas páginas usando o menu (assume arquivos `about.html`, `menu.html`, `reviews.html`, `contact.html` na mesma pasta).

## 🛠️ Customizações comuns

- Ajustar cores no `:root` de `style.css` (`--geral-color`, `--bg`, etc.).
- Trocar logo em `index.html` em `<img src="img/img-curta-ayvy.jpeg" ...>`.
- Adicionar conteúdo e seções nas páginas HTML referenciadas.

## 📌 Observações

- Hoje o projeto só tem `index.html` no repositório, mas o menu aponta para outras páginas que devem ser adicionadas para navegação completa.
- Ideal para protótipo de front-end ou prova de conceito de TCC.
